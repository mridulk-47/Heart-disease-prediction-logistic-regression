# Heart-disease-prediction-logistic-regression
A machine learning project that predicts the likelihood of heart disease using a Logistic Regression model. The model is trained and evaluated on a dataset containing various health parameters.

## Project Overview
This project demonstrates the development of a machine learning model to predict heart disease. Utilizing a Logistic Regression algorithm, the model analyzes various medical attributes to classify individuals as having or not having heart disease.

## Dataset

Dataset link= "https://drive.google.com/file/d/1CEql-OEexf9p02M5vCC1RDLXibHYE9Xz/view". 
It includes features such as age, sex, chest pain type, resting blood pressure, cholesterol, and more, with a 'target' variable indicating the presence (1) or absence (0) of heart disease.

## Methodology
1.  **Data Collection and Processing:** The data is loaded, inspected for missing values, and statistical summaries are generated.
2.  **Exploratory Data Analysis (EDA):** Initial analysis of the target variable distribution.
3.  **Data Splitting:** The dataset is divided into training and testing sets to evaluate model performance.
4.  **Model Training:** A Logistic Regression model is trained on the preprocessed training data.
5.  **Model Evaluation:** The model's accuracy is assessed on both the training and test datasets.
6.  **Predictive System:** A simple system is built to make predictions on new, unseen data.

## Results
The Logistic Regression model achieved an accuracy of approximately 85% on the training data and 81% on the test data.
